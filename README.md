# Landing Page - Como Entreter Seus Filhos Sem Celular

Landing page desenvolvida em React para promover o ebook **"Como Entreter Seus Filhos Sem Celular"**, que apresenta atividades e brincadeiras para ajudar pais a reduzir o uso de telas pelas crianças.

## Tecnologias utilizadas

- React
- Vite
- CSS
- JavaScript

## Objetivo do projeto

Criar uma página simples e atrativa para apresentar o ebook e direcionar os usuários para a página de compra.

## Como rodar o projeto

Você precisará ter o [Node.js](https://nodejs.org/) instalado em seu computador para rodar este projeto. Com o Node instalado, abra o terminal do seu computador (Prompt de Comando, PowerShell ou terminal do VS Code) e siga os passos abaixo:

1. **Clone o repositório:**
   No terminal do seu computador, digite o comando abaixo e aperte Enter:

```bash
git clone https://github.com/Reginaldo345/landing-page-entreter-filhos-sem-celular.git
```

2. **Entre na pasta do projeto:**
   Após o download dos arquivos terminar, acesse a pasta do projeto com o comando:

```bash
cd landing-page-entreter-filhos-sem-celular
```

3. **Entre na sub-pasta do código React:**
   Acesse o diretório específico onde o código React e as dependências estão:

```bash
cd landing-page-entreter-filhos-sem-celular-react
```

4. **Instale as dependências:**
   Ainda dentro dessa pasta, execute o comando abaixo para instalar todas as bibliotecas necessárias para o projeto rodar:

```bash
npm install
```

5. **Inicie o servidor de desenvolvimento:**
   Após a instalação ser concluída (pode levar alguns segundos), rode o comando abaixo para ligar o servidor local:

```bash
npm run dev
```

6. **Abra no seu navegador:**
   O terminal irá mostrar um link (geralmente `http://localhost:5173/`). Copie esse link, cole no seu navegador de internet e aperte Enter para visualizar a página funcionando.
